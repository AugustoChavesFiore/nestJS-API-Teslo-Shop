# Teslo API


1. Clonar Proyecto
2. ```pnpm install```
3. Clonnar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Cambiar variables de entonrno
5. Levantar la Base de datros
```
docker-compose up -d
```
6. Levantar ``` pnpm run start:dev```
